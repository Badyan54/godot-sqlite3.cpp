# godot-sqlite3-c++

Create project godot in demo. Download scons compiler and python.
clon repositori https://github.com/godotengine/godot-cpp.
Open terminal and cd in directori where Sconstruct and write "scons platform=<your_OS>".
make sure after compilation that the db_godot.gextension file is in the right bin
